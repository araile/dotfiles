# Ruby

## Install

 1. Install Bundler:

    ```sh
    gem install bundler
    ```

 1. Optionally, install the gems listed in [`Gemfile`](Gemfile):

    ```sh
    bundle install
    ```

## Configure

 1. Create a symlink to the [`irbrc.rb`](irbrc.rb) file
    (this can be run from any directory):

    ```sh
    ln -s .myconf/ruby/irbrc.rb $HOME/.irbrc
    ```
